### 21-09-16

-   ERROR: all, any 구분
    -   SITUATION: conditional situation
    -   REASON:
        -   all (all True)의 반대는 all False 가 아님
        -   any(any True) 의 반대는 any False가 아님
    -   SOLUTION:
        -   all = 모든 값이 True
        -   any = 하나라도 True
        -   not all = 모든 값이 False
        -   not any = 하나라도 False
