## TEL (Trial and Error Log)

### 21-05-21

-   ERROR: 서버 접속이 안됨

    -   SITUATION: local에선 잡히는데, 외부에서 연결이 안됨

    -   REASON: 방화벽 정책이 잡혀 있음 

    -   SOLUTION: 방화벽을 열어라 firewalld

        -   UFW 상태 확인

            -   ```shell
                sudo ufw status verbose
                ```

        -   port 추가

            -   ```shell
                sudo ufw allow 22 
                ```

        -   port 제거

            -   ```shell
                sudo ufw deny 22
                ```
